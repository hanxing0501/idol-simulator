✨ 偶像模拟器 · Idol Simulator

一个沉浸式的偶像养成模拟器，你可以在里面创建专属偶像、接收粉丝消息、发布动态、收集应援语，体验完整的偶像互动流程。

🌟 特色功能
🎤 **详细创建偶像**：填写姓名、艺名、年龄、风格标签、特长、粉丝名、代表色、出道日期、背景故事、外貌性格等，打造独一无二的角色。
💬 **实时消息互动**：接收粉丝私信、工作邀约，模拟真实的偶像日常。未读消息红点提示，支持标记已读。
📝 **动态发布**：随时发布动态与粉丝互动，自动生成粉丝评论和点赞。
💜 **应援墙**：粉丝为你撰写应援语，基于偶像信息动态生成，每次打开都有新鲜感。
📊 **数据看板**：粉丝数、热度值、动态数一目了然，数据会随时间和互动自动增长。
📱 **iOS 风格界面**：精心还原 iOS 的卡片设计、毛玻璃导航、标签栏和交互细节，在手机和电脑上都能获得原生感体验。
💾 **本地存储**：所有数据自动保存到浏览器，刷新不丢失。
🌐 **一键部署**：纯前端静态应用，可部署至 GitHub Pages、Netlify、Vercel 等，免费获得 HTTPS 访问。
